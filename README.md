# Draft Lab

Draft Lab es una aplicación web completa para la gestión y visualización de *drafts* en torneos de juegos MOBA. Permite a equipos de producción y *staff* controlar en tiempo real *picks*, *bans* y estados de partida, generando *overlays* listos para transmisión en herramientas como OBS.

La plataforma está diseñada para entornos competitivos, tanto comunitarios como profesionales, ofreciendo una interfaz intuitiva para operadores y la capacidad de integrarse con plataformas externas como **Battlefy** y **Toornament** mediante *scraping* de datos.

## 🚀 Características Principales

* **Control en tiempo real:** Gestión instantánea de *picks*, *bans* y estado de la partida.
* **Overlays para streaming:** Generación de gráficos listos para integrar directamente en OBS y otras herramientas de transmisión.
* **Interfaz intuitiva:** Diseñada específicamente para facilitar el trabajo ágil y sin errores de los operadores durante torneos en vivo.
* **Integración externa:** Sincronización ágil de datos mediante *scraping* de plataformas líderes en gestión de torneos.
